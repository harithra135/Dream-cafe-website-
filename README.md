# Dream Cafe

Dream Cafe is a simple multi-page café website developed using HTML and CSS. The project is designed to provide an attractive and user-friendly interface where visitors can explore the café, view the menu, place an order through a form, browse the gallery, read customer reviews, and find contact information.

This project was created as part of a Web Designing assignment to demonstrate the fundamentals of HTML and CSS.

## Features

- Home page with a welcome section
- Menu page displaying food and beverages
- Order page with an order form
- Gallery page with café images
- Reviews page with customer feedback
- Contact page with café details and contact form
- Responsive layout
- CSS animations and hover effects
- Simple and clean user interface

## Technologies Used

- HTML5
- CSS3
